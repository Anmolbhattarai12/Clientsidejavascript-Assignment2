The Pizza Maker

This is my Assignment 2 project for COMP1073 Client-Side JavaScript.

The project is a simple pizza order form. The user can enter their name and choose pizza size, crust type, sauce type, cheese, toppings, and special instructions.

When the form is submitted, JavaScript validates the form, captures the form values, creates a Pizza object using a Pizza class, and displays the pizza order on the page using a method from the Pizza object.

Features:
- Pizza order form
- Form validation
- Student name and ID added using JavaScript
- Pizza class
- Pizza object created from form values
- Pizza description displayed using a class method
- Simple Nepali theme
- Pizza image

Files:
- index.html
- css/styles.css
- js/script.js
- img/pizza.jpg

How to Use:
1. Open the website.
2. Fill in the pizza order form.
3. Select pizza options.
4. Click Order Pizza.
5. The order description will appear on the page.
